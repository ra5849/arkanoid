# Proyecto Arkanoid

Juego HTML5 en un solo archivo: sin sistema de compilación ni dependencias.

## Archivos
- `ark.html` — juego autónomo completo (sin dependencias externas excepto Google Fonts)

## Pruebas
- Abrir `ark.html` directamente en cualquier navegador moderno
- No hacen falta comandos de compilación, lint ni tests

## Notas clave
- El juego usa Web Audio API para síntesis de sonido (sin archivos de audio)
- Puntuaciones altas guardadas en localStorage del navegador
- Toda la lógica, estilos y recursos están inline en el único HTML
